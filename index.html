Unit 1 : Introduction To HTML

  1. Write a program to display Paper Code with Paper Name using an HTML table with six rows and two columns.
  2. Write a program to align an entire HTML table to right.
  3. Write a program to align within a row of HTML table.
  4. Write a program to align within a cell of HTML table.
  5. Write a program to use HTML Summary Attribute With Table Tag.
  6. Write a program to use Caption Tag.
  7. Write a program to set bgcolor attribute for HTML table.
  8. Write a program to set width and height attributes for HTML table.
  9. Write a program to add border of HTML table.
  10. Write a program to use cellspacing and cellpadding attributes for HTML table.
  11. Write a program to use colspan and rowspan attributes for HTML table.
  12. Write a program to use thead, tbody, and tfoot tags for HTML table.
  13. Write a program to use colgroup tag for HTML table.
