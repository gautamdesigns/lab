Web Technology Lab

  1. Create a static website for your family. The home page should contain a suitable name of your family, introduction, postal address, common mobile/landline number and a picture of the family members. Different heading tags and paragraphs in different face, size, and color of fonts should be applied wherever necessary. Apply all necessary attributes of the image tag to the picture of the family members. The names of each family member should be added at the bottom of the corresponding picture of the family members, where each of the names should link to a personal page of the corresponding person, i.e., there should be a separate personal page about each of the family members including personal information such as name, personal mobile number, email id, date of birth, hobbies and favourite pastimes, etc. Also use the picture of the family members as image maps, where each of the pictures should link to the same personal page of the corresponding person.
  2. Create an HTML document to display an ordered list of your top five favourite movies. Each element of the list must have a nested unordered list of songs with the name of the vocal artist. You may apply text formatting tags to the lists.

  5. Write a JavaScript function to print the contents of the current window.
  6. Write a JavaScript program to get the current date. Expected Output: mm-dd-yyyy, mm/dd/yyyy, dd-mm-yyyy and dd/mm/yyyy.
  7. Write a JavaScript function to find the area of a triangle where the lengths of the three of its sides will pass through the HTML page.
  8. Write a JavaScript program where the program takes a random integer between 1 and 10, the user is then prompted to input a guess number. If the user input matches with guessed number, the program will display the message “Good Work” otherwise, display a message “Not matched”.
  9. Write a JavaScript function that reverses the digits of a number.
  10. Write a JavaScript function that returns a passed string with letters in alphabetical order.
  11. Write a JavaScript function that accepts a sentence and converts the first letter of each word of the sentence in upper case.
  12. Write a JavaScript function that accepts a sentence and counts the number of vowels within that sentence.
  13. Write a JavaScript function to compute the value of bn where 'n' is the exponent and 'b' is the base. Accept 'b' and 'n' from the user and display the result.
